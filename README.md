This web application leverages Natural Language Processing (NLP) and Machine Learning (ML) techniques to streamline the recruitment process by acting as an intelligent resume filter. Built with Python Flask on the backend and HTML/CSS for the frontend, the system automatically analyzes job descriptions and matches them against a pool of available resumes.
The application extracts key skills, qualifications, and experience requirements from job postings, then compares them with candidate resumes using NLP-based text similarity and classification methods. By doing so, it helps recruiters quickly identify the most relevant candidates, reducing manual effort and improving efficiency in talent acquisition.
With its user-friendly web interface, recruiters can upload job descriptions and resumes, view ranked matches, and make data-driven hiring decisions. The integration of Flask ensures smooth server-side processing, while HTML and CSS provide a clean and responsive design for seamless user interaction.
This project demonstrates the practical application of NLP and ML in solving real-world HR challenges, offering a scalable solution for resume screening and candidate-job matching.
